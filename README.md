# CryptoSniperBot

Автоматизированная система мониторинга рынка криптовалют и генерации торговых сигналов.

## Описание проекта
Проект представляет собой Python-бота, который анализирует рынок в реальном времени, используя стратегию Smart Money. Бот самостоятельно отслеживает рыночные данные, применяет технические индикаторы и отправляет уведомления о точках входа с рассчитанным риск-менеджментом.

## Основные возможности
- **Анализ в реальном времени:** Асинхронная обработка данных для высокой скорости отклика.
- **Технический индикатор:** Анализ зон ликвидности (FVG), MACD, RSI.
- **Риск-менеджмент:** Автоматический расчет Stop-Loss и Take-Profit для каждой сделки.
- **Гибкие настройки:** Конфигурация стратегий для разных торговых пар через `best_strategies.json`.

## Технологии
- **Язык:** Python
- **Методология:** AI-assisted development (разработка с помощью ИИ)
- **Инструментарий:** Asyncio, Pandas, JSON-конфигурация
